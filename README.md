# Cal AI Clone - AI Calorie Tracker

An AI-powered mobile app that analyzes food photos and provides instant nutritional breakdowns including calories, macros, and dietary insights.

## 🎯 Project Overview

This is a challenge submission for building a Cal AI clone - one of the highest-revenue AI health apps on the App Store. Users can photograph their meals and receive AI-powered nutritional analysis with daily tracking, macro goals, and streak tracking.

## ✨ Key Features

- 📸 **Photo Recognition** - Snap a photo or upload to get instant food recognition
- 🤖 **AI-Powered Analysis** - Get accurate calorie, protein, carbs, and fat breakdown
- 📊 **Daily Food Diary** - Track all meals with running calorie totals
- 🎯 **Macro Progress Rings** - Visual indicators for macro goals (protein/carbs/fat)
- 📈 **Weekly History Charts** - Visualize nutritional trends over time
- 🔥 **Streak & Goal Tracking** - Stay motivated with streak counters and custom goals
- 💾 **Persistent Storage** - Save and review your entire food history
